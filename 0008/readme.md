## HTML_Extractor

**基于行块分布函数的通用网页正文抽取算法**

算法主要原理

1.正文区密度：在去除HTML中所有tag之后，正文区字符密度更高，较少出现多行空白；

2.行块长度：非正文区域的内容一般单独标签（行块）中较短。

具体实现方法参考原作者陈鑫的PDF
