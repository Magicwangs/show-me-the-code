## assert
`assert`断言是声明其布尔值必须为真的判定，如果发生异常就说明表达示为假。可以理解assert断言语句为raise-if-not，用来测试表示式，其返回值为假，就会触发异常。

assert断言是一句必须等价于布尔真的判定！
如果断言成功（如果为真）那么不执行任何操作！
如果断言不成功，那么会触发AssertionError
